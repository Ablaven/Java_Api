4. Fetch & JSON ‚ñº

What is Fetch? - The Messenger

fetch() is like sending a text message to the backend:

const response = await fetch('api.php?endpoint=business_data');

Step by step:

  1. Send message to api.php asking for business data
  2. Wait for response (that's what await does)
  3. Receive answer from backend
  4. Convert answer from JSON text to JavaScript objects

What is JSON?

JSON is like a universal language for computers:

{
    "id": 1,
    "name": "John",
    "salary": 50000
}

It's just text that follows specific rules, like writing addresses on envelopes.

5. Backend (PHP) ‚ñº

üç≥ Backend - The Kitchen (api.php)

PHP runs on the server (the computer hosting the website), not in user's browser.

What is an API Endpoint?

An API endpoint is like a specific phone number for different services:

  • api.php?endpoint=employees = "Call employee department"
  • api.php?endpoint=products = "Call inventory department"
  • api.php?endpoint=analytics = "Call statistics department"

How PHP Works:

<?php  ‚Üê Start PHP code
// Everything between these tags runs on server

$endpoint = $_GET['endpoint'] ?? 'business_data';

Breaking this down:

  • $endpoint = Create a variable (box) called "endpoint"
  • $_GET['endpoint'] = Look at the website address for ?endpoint=something
  • ?? = "If that doesn't exist..."
  • 'business_data' = "...use this default value instead"

Database Connection - Plugging Into Storage

$pdo = new PDO("mysql:host=$host;dbname=$dbname;charset=utf8", $username, $password);

What this does:

  • PDO = PHP Data Objects (a tool for talking to databases)
  • mysql:host=localhost = "Connect to MySQL database on this computer"
  • dbname=dashboard_db = "Open the 'dashboard_db' storage room"
  • charset=utf8 = "Use international character encoding"

What is MySQL?

MySQL is like a giant, organized filing cabinet:

  • Database = The entire filing cabinet
  • Tables = Individual drawers
  • Columns = Categories in each drawer (Name, Age, etc.)
  • Rows = Individual records (one person's information)
[Image of relational database table structure diagram]

SQL Commands:

CREATE TABLE employees (      ‚Üê Make a new drawer called "employees"
    id INT AUTO_INCREMENT PRIMARY KEY,  ‚Üê Automatic numbering system
    name VARCHAR(100) NOT NULL,          ‚Üê Text field, required
    salary DECIMAL(10,2) NOT NULL        ‚Üê Number with 2 decimal places
);

Field Types Explained:

  • INT = Whole numbers (1, 2, 3...)
  • VARCHAR(100) = Text up to 100 characters
  • DECIMAL(10,2) = Numbers like 12345678.99 (10 total digits, 2 after decimal)
  • DATE = Dates like 2024-01-15
  • TEXT = Long text (like paragraphs)

8. Lifecycle, Security & Debugging ‚ñº

üîÑ How Everything Works Together

[Image of full stack web application data flow diagram]

The Complete Journey:

  1. User opens website - Browser downloads index.html - HTML tells browser to also get style.css and script.js - Chart.js library downloads from internet
  2. Page loads - script.js runs: document.addEventListener('DOMContentLoaded', function() {...}) - This means: "Wait until page is fully loaded, then do this"
  3. Data loading begins - JavaScript calls loadData() function - fetch('api.php?endpoint=business_data') sends request to server
  4. Server processes request - Apache web server receives request - Passes it to PHP interpreter - PHP code in api.php runs - PHP connects to MySQL database - Executes SQL query: SELECT id, date, category, value, note FROM business_data - Database returns rows of data - PHP converts data to JSON format - Sends JSON back to browser
  5. Browser receives data - JavaScript receives response - response.json() converts JSON text to JavaScript objects - updateTable(data) puts data into HTML table - updateBusinessCharts(data) draws charts
  6. User interaction - User clicks cell in table - contentEditable="true" allows editing - When user clicks elsewhere: blur event fires - JavaScript sends UPDATE request to api.php - PHP updates database - Charts refresh with new data

üîß Key Programming Concepts

Asynchronous Programming (async/await):

async function loadData() {         ‚Üê This function can wait for things
    const response = await fetch('api.php');  ‚Üê Wait for server response
    const data = await response.json();       ‚Üê Wait for JSON conversion
}

Why this matters: Internet requests take time. Without await, JavaScript would continue before getting response.

üîê Security Features

Prepared Statements:

$stmt = $pdo->prepare('UPDATE business_data SET value = :value WHERE id = :id');
$stmt->execute([':value' => $newValue, ':id' => $recordId]);

Prevents SQL injection attacks where hackers try to run malicious database commands.
